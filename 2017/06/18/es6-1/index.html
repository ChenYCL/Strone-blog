get和set方法

static 静态方法 不需要实例化就可以使用的方法 用类名.方法名 就行了

extends 继承属性

Set

集合类似,不能重复定义，里面的值不能包含重复的东西，如果是重复的东西，集合中只会有一个，可用这点查重判断 要添加可用 add方法,要获取集合的长度 使用 size,判读某个值是否在里面用has 方法 delete 删除，循环处理用forEach(),清空用 clear（）方法
